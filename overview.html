OVERVIEW

The graph shows the data gathered from the device in real time. To read the data easier, the graph is scaled down to the following ratios:

In temperature, 1 unit is equivalent to 10 degrees Celsius.

For Total Dissolved Solids (TDS), 1 unit is equivalent to 100 ppm.

Turbidity is also scaled with 1 unit being equivalent to 10 ntu.

pH value is not scaled, as the value only reaches a maximum of pH 14.

For clarity, users can hover their mouse over a specific point in the graph, which will show the scaled value and the real value of each component.
